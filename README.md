# Engineering Portfolio | Joseph Kothapalli

This repository contains the architecture files for my single-page responsive software engineering portfolio. Built utilizing an uncompromised structural data modular separation scheme.

## 🛠️ Tech Architecture Matrix
- **Core Layout Structure:** HTML5 Semantic Engine
- **Style Matrix Processing:** Native CSS Grid & Custom Property Flex Systems
- **Dynamic Content Injection:** JavaScript ECMAScript DOM Orchestrator
- **Typography Layouts:** Google Fonts (Space Grotesk & Fira Code)

## 📂 Editing Portfolio Data
To modify biographical info, adjust core competencies, or load new technical projects:
1. Navigate directly to `updates/data.js`.
2. Alter properties inside the `PORTFOLIO_DATA` object model.
3. Save file—changes render natively across target nodes instantly without layout modifications.
